# 投影片說明 / Slides

本資料夾存放 Vibe Coding 工作坊各日的投影片 HTML 檔案。

## 檔案清單

| 檔案 | 說明 |
|------|------|
| `day1.html` | Day 1 投影片：AI 開發概念、prompt 技巧、第一個 Claude Web App |
| `day2.html` | Day 2 投影片：Claude Code 本地開發、機器學習管線實作 |

## 使用方式

直接用瀏覽器開啟 HTML 檔案即可。按下 **← →** 方向鍵或點擊左右側切換投影片頁次。

- 講義連結（配合投影片閱讀）：`../day1.html`、`../day2.html`
