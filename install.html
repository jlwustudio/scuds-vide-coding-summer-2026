Day 2 環境安裝指南

Day 2（7/19）要用 Claude Code——能直接讀寫你電腦檔案的 AI。下面兩種方式擇一裝好就能跟課。不確定選哪個，就選方式 A。

Part 0・開始之前：你需要這三樣

先確認手上有這些，再往下走。缺哪一個先補哪一個。

要有什麼 怎麼確認
你自己的筆電 Windows 或 Mac 都可以。記得帶充電器。
Claude Pro 帳號 打開 claude.ai 登入，確認左下角帳號顯示為 Pro。沒有 Pro 請課前聯絡助教，現場處理會吃掉你的實作時間。
15 分鐘與網路 裝在家裡網路穩的地方，不要等到上課現場才裝。
⚠️ 重要

方式 A 和方式 B 只要裝一種。兩種是同一顆引擎、同樣的功能，差別只在「按按鈕」還是「打指令」。不用兩種都裝。

Part A・認識 Claude Code：它跟網頁版差在哪

Day 1 我們在瀏覽器裡跟 Claude 講話，它把成品做在右邊的小視窗裡——你關掉分頁它就沒了，也沒辦法接上其他程式。

Day 2 升級：Claude Code 把 AI 搬進你的電腦。它能直接讀寫你的檔案、真的執行程式。做出來的東西是存在你硬碟裡的真專案，不是瀏覽器裡的預覽。

💡 白話比喻

網頁版像是隔著玻璃跟實習生講話，他做好東西舉起來給你看。Claude Code 是讓實習生直接坐到你的電腦前面動手——所以它每次要改你的檔案，都會先問過你。

兩種方式怎麼選

🖥️ 方式 A：Desktop App ⌨️ 方式 B：CLI 終端機
適合誰 沒用過終端機的同學（推薦） 已經習慣打指令、想體驗純文字操作
怎麼操作 全程用點的，有圖形介面 全程打指令
要裝幾個東西 一個 App（Windows 另需 Git） 一行安裝指令
安裝失敗率 較低 較高

課堂全程主要示範方式 A，方式 B 的對照寫在旁邊，兩種都能跟課。

Part B・選一種方式裝好並登入

先選一下你的系統，下面的步驟會跟著變：

🖥️ 方式 A：Claude Desktop App（推薦）

一個應用程式裝好就能用，不需要另外裝終端機工具，也不需要 VS Code。

1

下載

前往 claude.com/download，頁面上點你系統對應的按鈕：

👀 你會看到：一顆寫著 Download for macOS 的按鈕。點它。
👀 你會看到：一顆寫著 Download for Windows 的按鈕。點它。
（如果你的電腦是 ARM 版 Windows，才選 Windows (arm64)；一般筆電不用管這個。）
2

安裝

  1. 下載完成的 .dmg 檔案雙擊打開
  2. 把 Claude 圖示拖進 Applications 資料夾
  3. 從 Launchpad 或應用程式資料夾打開 Claude
🆘 如果 Mac 說「無法打開，因為無法驗證開發者」

在 Claude 圖示上按右鍵 → 打開，再點一次「打開」就會過。（這是 macOS 對新下載 App 的例行攔截，不是出錯。）

  1. 下載完成的安裝檔雙擊執行
  2. 若跳出「你要允許這個應用程式對你的裝置進行變更嗎」，點「是」
  3. 依照安裝精靈完成，全部保持預設值即可
3

登入

開啟 Claude 後，用你的 Claude Pro 帳號登入。

👀 你會看到：它會跳出瀏覽器頁面讓你完成登入，登入後會自動跳回 App。
4

Windows 專屬：先裝 Git

⚠️ 這步不能跳過

在 Windows 上，Code 分頁需要 Git 才能啟動。沒裝的話下一步點下去會沒反應。
（Mac 多半已內建 Git，所以 Mac 沒有這步。）

  1. 到 git-scm.com/install/windows
  2. 點頁面上的 「Click here to download」（或 Standalone Installer 底下的 Git for Windows/x64 Setup）
  3. 雙擊下載好的檔案，執行安裝精靈，一路點「Next」保持預設值，最後點「Install」
⏱ 安裝約 3 分鐘，全程圖形化操作，不需要打任何指令。裝完不用重開機。
45

打開 Code 分頁

登入後，點 App 上方的 Code 分頁。

👀 你會看到：一個可以選資料夾、跟 Claude 對話的畫面。
如果跳出升級或再次登入的提示，照著做完再點一次 Code。
✅ 方式 A 完成確認

Claude Desktop 開得起來、登入成功、點得到 Code 分頁而且沒有卡在升級或登入畫面——三個都對就完成了，跳到 Part C。

⌨️ 方式 B：Claude Code CLI（進階）

已經用習慣終端機、或想體驗純指令操作的同學再選這個。裝了方式 A 的同學不用做這段。

1

打開終端機

打開「終端機」（Terminal）——按 ⌘ + 空白鍵 叫出 Spotlight，搜尋 terminal 就找得到。

打開 PowerShell——點開始選單，搜尋 powershell 就找得到。

2

貼上安裝指令

複製整行貼上，按 Enter：

curl -fsSL https://claude.ai/install.sh | bashirm https://claude.ai/install.ps1 | iex
👀 你會看到：下載進度，最後顯示安裝完成的訊息。約 1–2 分鐘。
3

開一個新的終端機視窗

⚠️ 這步很容易漏

安裝完成後，原本開著的視窗還讀不到新指令。請把它關掉、重開一個新的。不做這步，下一步一定會說找不到指令。

4

啟動並登入

在新視窗輸入：

claude

首次啟動會引導你登入——選擇 Claude account（用你的 Claude Pro 帳號），瀏覽器會跳出頁面完成授權，授權完回到終端機就能繼續。

確認可以對話之後，輸入 /exit 離開。

✅ 方式 B 完成確認

claude --version 有版本號、claude 能進入對話畫面——兩個都對就完成了。

Part C・兩種方式都需要：Python 3

資料科學系同學多半已經有了。先確認一下，在終端機輸入：

python3 --versionpython --version
👀 有顯示版本號（例如 Python 3.12.4）就代表有了，這段跳過。

沒有的話，從 python.org/downloads 下載安裝。

⚠️ Windows 安裝 Python 最重要的一步

安裝精靈第一頁底下有個小勾選框 「Add python.exe to PATH」——一定要勾。沒勾的話終端機會找不到 python，要重裝一次才能補救。

（VS Code 只有選方式 B 的同學需要另外裝：code.visualstudio.com。Desktop App 本身就內建檔案編輯器，選方式 A 的不用裝。）

進階選配（非必要）

Day 2 有「加分挑戰」，其中一項是體驗 Python 虛擬環境（venv）。這不是必修，沒裝也完全不影響當天任何流程，行有餘力再看就好。

FAQ・卡住了？

💡 先試這招

把錯誤訊息整段複製，貼到 claude.ai 問它。這不是偷懶——這就是 vibe coding 的第一課，你已經在做了。

Windows：點了 Code 分頁沒反應

九成是沒裝 Git。回到 Part B 步驟 4 裝好 Git，再重開 Claude Desktop。

App 裡根本找不到 Code 分頁

版本太舊。Mac：點左上角選單 Claude → Check for Updates。Windows：Help → Check for Updates。更新後重啟 App 再看一次。

登入了，但顯示的是免費版不是 Pro

先到 claude.ai 網頁版確認左下角是不是 Pro。如果網頁版是 Pro 但 App 不是，在 App 登出再登入一次。還是不行請課前找助教——現場處理會吃掉你的實作時間。

CLI：裝完之後打 claude 說「command not found」

最常見原因是沒有開新的終端機視窗。舊視窗讀不到剛裝好的指令。關掉、重開一個，再試一次。

重開還是不行的話，建議直接改用方式 A（Desktop App），不要在這裡耗時間。

Windows：打 python3 說找不到

Windows 上通常是 python 而不是 python3，先試 python --version。

如果兩個都找不到，多半是安裝 Python 時沒勾「Add python.exe to PATH」。重跑一次 Python 安裝檔、記得勾那個框。

Mac：說「無法打開，因為無法驗證開發者」

在 App 圖示上按右鍵 → 打開，再點一次「打開」就會過。這是 macOS 對新下載 App 的例行攔截，不是出錯。

安裝跑到一半好像卡住不動了

先等滿 3 分鐘——下載大檔案時進度條不動是正常的。超過 3 分鐘完全沒變化，關掉重跑一次。連續兩次都卡在同一個地方，換另一種方式裝（A 換 B 或 B 換 A）。

兩種方式都裝了會不會衝突？

不會衝突，但沒必要。它們是同一顆引擎的兩種介面。挑一個你用得順的就好。

我的筆電是公司／學校的，沒有管理員權限

先照方式 A 試試看，很多情況下 Desktop App 裝得起來。真的被擋，請課前告訴助教，我們會安排你當天跟鄰座併桌共看一台。不要拖到上課才說。

我試了半天真的裝不起來，明天怎麼辦？

不用慌，一定有得跟。課堂備有三層救援：換另一種安裝方式、跟鄰座同學併桌共看一台、或改用 Day 1 的網頁版跟課（講義附有對照表）。

但自己的環境能動收穫最大，所以：明早 08:45 前到教室，助教會一對一幫你處理。
